## System Requirements

- A modern web browser: Chrome 120+, Firefox 121+, Edge 120+, or Safari 17+
- A stable internet connection
- For 2FA: a smartphone with Google Authenticator or Microsoft Authenticator installed

No software installation is required. The system runs entirely in the browser.

---

## Getting Help

If you encounter a problem not covered in your manual:

1. Check the **Common Problems** section at the end of your role's manual
2. Contact your school administrator
3. For technical issues with the system itself, the administrator can refer to the project repository at https://github.com/Khaalid245/somali-early-warning-system-school
